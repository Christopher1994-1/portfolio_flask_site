About This Site

I developed a website using Python's Flask web framework. Like the UFO dataset website, this website also deals with another large dataset of sightings. But this one deals with a lot more data per individual row. I organize the data in easy-to-read ways and make it easy for the user to navigate around and find what they are looking for.




What skills does this project showcase?

  • Web Development: I have developed a website using Flask, which is a web framework that uses Python.
  • Data Analysis and Visualization: I have used a bigfoot sightings dataset to create interactive maps, charts, and tables that display information about the sightings in an easy-to-read and navigate format.
  • Data Filtering and Sorting: I have implemented features that allow users to filter and sort the data based on different criteria, such as date, location, or type of sighting. User Interface Design: I have designed a user-friendly interface that makes it easy for users to navigate and interact with the data displays and features.
