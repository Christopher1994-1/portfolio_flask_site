About This Site

Introducing my demo website built on Python's Flask web framework! This demo template is designed for a marijuana dispensary and offers a variety of impressive features to provide a seamless user experience.

With multiple SQLite3 databases, the site can easily store and retrieve information on products, reviews, and admin details. Plus, with a handy email subscription feature, users can stay up-to-date on the latest dispensary news and products.

The website's visually stunning design is achieved with HTML, CSS, and JavaScript, creating an intuitive and user-friendly interface. Its various functions are sure to impress and make for an exceptional user experience.




What skills does this project showcase?

  • Python: A good understanding of Python programming language, including its syntax, data structures, and object-oriented programming principles.
  • Flask Framework: Knowledge of Flask web framework, including how to create and manage routes, handle HTTP requests and responses, and work with templates.
  • HTML/CSS: Familiarity with HTML and CSS for creating web pages and styling them.
  • JavaScript: Knowledge of JavaScript for adding interactivity to web pages, such as forms validation and dynamic page updates.
  • SQLite: Understanding of SQLite database management system and how to integrate it with Flask to store and retrieve data.
  • Web Development: Experience using development tools like Git for version control, text editors or integrated development environments (IDEs) for coding, and web debugging tools for testing and troubleshooting.
  • UI/UX Design: Basic understanding of user interface and user experience design principles, including how to create responsive web designs that work across different devices and screen sizes.
