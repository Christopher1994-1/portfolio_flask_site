About This Site

I developed a website using python flask web framework, it is a UFO sightings website that uses a dataset of over 117,000 rows and distributes the data in such a way that a user can look through the latest sightings, monthly sightings, statistics on the dataset, and even search for a location to see the relevant data in that area.




What skills does this project showcase?

  • Web Development: Using the Flask web framework and building a full-featured website demonstrates my knowledge of web development principles and technologies, as well as my experience with a popular web development framework.
  • Data Analysis & Manipulation: Working with a dataset of over 117,000 rows and distributing the data to provide relevant information for a user demonstrates my ability to analyze and manipulate data, extract insights, and present that information in a user-friendly manner.
  • User-Centered Design The ability to search for location-based information and view the latest and monthly sightings showcases my understanding of creating user-friendly and functional websites that meet the needs of users.
  • Problem-Solving: Developing a website that can handle and process a large dataset to provide relevant information demonstrates my ability to identify a problem and find a solution through technology.
