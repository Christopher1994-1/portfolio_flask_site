About This Site

I developed a comprehensive website using the Django framework, integrating the Open Weather API and Geo-Names API to deliver real-time weather information for user-specified cities. In addition to providing current weather conditions, the website has the capability to function as a news aggregator, offering a diverse range of news sources and updates. My project showcases my strong skills in web development and API integration, as well as my ability to create functional and user-friendly websites




What skills does this project showcase?

  • Web Development: Building a comprehensive website using the Django framework demonstrates my knowledge of web development principles and technologies, as well as my experience with a popular web development framework.
  • API Integration: Integrating the Open Weather API and Geo-Names API demonstrates my ability to work with APIs, access and retrieve data from external sources, and integrate that data into my web application.
  • User-Centered Design: The capability of the website to function as a weather and news aggregator showcases my understanding of creating user-friendly and functional websites that meet the needs of users.
  • Problem-Solving: Developing a project that integrates APIs and provides real-time weather information for user-specified cities demonstrates my ability to identify a problem and find a solution through technology.
